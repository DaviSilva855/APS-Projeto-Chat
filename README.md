# APS-ChatSocket

## Sobre o Projeto<br>
- Este projeto foi desenvolvido como uma Atividade Prática Supervisionada (APS) e tem como objetivo implementar um sistema de chat em rede utilizando sockets e threads, permitindo a comunicação entre múltiplos usuários em tempo real através de um servidor central.<br><br>
- O sistema foi desenvolvido em Python e executado via terminal, utilizando comunicação TCP/IP local.

## Tecnologias Utilizadas

- Python
- Socket
- Threading

## Como Executar

- Execute primeiro o arquivo servidor.py pelo terminal:

- Informe uma porta para iniciar o servidor (ex: 10000)

- Em seguida, execute o arquivo cliente.py em outro terminal:

- Informe a mesma porta do servidor
  
- Digite o nome de usuário

- Para adicionar mais usuários, basta executar cliente.py em novos terminais
